Odoo session

2017年2月18日
| Odoo

用户长时间不操作，自动退出。 通过修改部分源码，勉强可以工作。

client:

Path: D:\odoo\src\addons\web\static\src\js\framework\ajax.js 添加下面代码:

if(result.error.message == "Odoo Session Expired"){
    window.location.href = '/';
}

server:

path: D:\odoo\src\odoo\http.py
